# Target Audience

An analysis of the content's primary audience with end-user personas.

Coveo end users can be technical or non-technical—the goal is to tailor UX writing to the least experienced members of the audience.
Copy should make the next step obvious, use plain language, and help all users act without having to learn the product first.
