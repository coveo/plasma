To avoid confusion and ease the learning process for Coveo end users, it's crucial to choose one consistent term for each product, feature, and industry concept.

This glossary lists the official spelling, capitalization, and punctuation of the Coveo product terminology to be used in the UI.

> **Note:** The Product Documentation and other Coveo sources don't all have the same standardized terminology.
> Always refer to this glossary when writing for the UI.

## Banned terminology

Use this list to see which words you **can't** use when writing for the UI.
Go for their correct counterpart instead.

| Banned term | Replace with |
| ----------- | ------------ |
| Auth | Authentication |
| Admin console, Admin UI, Admin portal | Administration Console |
| c4sf, cfsf | Coveo for Salesforce |
| c4sc, cfsc | Coveo for Sitecore |
| Custom login | Custom authentication |
| HIP | Hosted Insight Panel |
| HSP | Hosted Search Page |
| KB content | Knowledge Base content |
| Log in, Log out | Sign in, Sign out |
| Meta data | Metadata |
| NPM | Node Package Manager |
| Pop up, Popup, Pop-up | Modal |
| UID, User ID | Client ID |

## Missing terms
