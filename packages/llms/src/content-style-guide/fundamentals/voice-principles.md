# Voice Principles

Good UX copy starts with knowing the difference between voice and tone, and then knowing how to apply each one.

## Understanding *voice* and *tone*

**Voice** is constant. It's Coveo's personality, staying consistent no matter where copy appears in the product.

**Tone** is situational. It shifts to match the UI context and emotional state of the user in that moment.

Think of it this way: whether you're explaining a complex idea or celebrating an achievement, you're still yourself. Your voice doesn't change, but you adapt your tone to fit the situation.

| Example | Coveo voice | Non-Coveo voice |
|---------|-------|-----------|
| **A user is deleting a query pipeline association** | You're about to delete "association-123". This action can't be undone. | Say goodbye to "association-123"? It'll be gone for good! |
| **A user checks their watchtower for errors** | Nothing to report. If something comes up, it'll appear here. | *Crickets...* No news is good news. If something breaks, we'll post it right here. |

| Example | Direct tone | Enthusiastic tone | Calm & instructional tone |
|---------|--------|--------------|----------------------|
| **A user is deleting a pipeline** | You're about to delete "pipeline-123". This action can't be undone. | Success! Your query pipeline "pipeline123" has been saved. | No query pipelines found. Try changing or clearing your filters. |

## What is Coveo's UX voice?

Coveo's voice can be described with three main qualities which should be present in every piece of copy.

### Clear

Copy is easy to understand on the first read. It uses plain words, gets to the point, and respects the user's time. When something is complex, clear copy breaks it down.

Coveo copy sounds like a knowledgeable colleague explaining something, not an instruction manual.

### Human

Copy speaks to the user directly. It uses natural language, avoids corporate stiffness, and treats users like capable adults. It's warm without being overly casual, and clear without being cold.

Coveo copy sounds like it was written by a person, for a person.

### Helpful

Copy focuses on what the user needs to do and why it matters to them. It answers the question before the user has to ask it, and it guides without hovering.

Coveo copy earns trust by being genuinely useful.
