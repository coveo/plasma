"Content" refers to all text, images, and videos in the Administration Console interface that help users navigate the product.

## Audience

These guidelines apply to anyone writing, editing, or reviewing UX content: designers, developers, and technical writers.

## Glossary

**Content guidelines** — Rules for keeping UX content clear, consistent, and on brand.

**Microcopy** — Short, functional text at decision points: tooltips, placeholder text, error messages, helper text.

**Short-form copy** — Brief, scannable text communicating a single idea. Includes page titles, panel descriptions, confirmation text, and banners.

**UI copy** — Text rendered inside a UI component: button labels, menu items, field labels, status messages.

**UX copy** (also *UX writing*) — All language within an interface that guides users through their experience.
